# TokenAuth
Token Authentication


## API request example
1. Register example
```json
{
    "username": "newuser",
    "password": "password123"
}
```
2. Login example
```json
{
    "username": "newuser",
    "password": "passwrod",
    "token": "your_generated_token_here"
}
```
3. Logout example
```json
{
    "message": "Logged out successfully"
}
```
